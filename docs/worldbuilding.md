# Worldbuilding

## 세계의 기원

태초의 **원초의 흐름(Primordial Flow)** 에서 세계가 탄생했습니다. 이 흐름이 응축된 곳마다 서로 다른 이름이 붙었습니다:

| 신화 체계 | 흐름의 이름 |
|-----------|-------------|
| 그리스 | 올림포스의 화로 |
| 북유럽 | 위그드라실의 뿌리 |
| 동아시아 | 쿤룬의 봉우리 |
| 이집트 | 두아트의 강 |
| 메소아메리카 | 테오티우아칸의 태양 |
| 켈트 | 티르 나 노그의 안개 |

모든 신화는 같은 기원의 서로 다른 기록입니다.

## 신들과 봉인

흐름에서 태어난 존재들은 스스로를 "신"이라 불렀으나, 흐름의 법칙을 거스를 수는 없었습니다. 신들은 둘로 나뉘었습니다:

- **창조자** — 생명의 씨앗을 뿌리고, 힘을 별자리의 좌표에 저장
- **파괴자** — 씨앗을 삼키고 왜곡을 불러오려는 자들

이들의 충돌이 **만신전쟁(War of Ten Thousand Gods)** 을 일으켰습니다.

전쟁 이후 신들은 사라지지 않았습니다. 그 본질은 **봉인**과 **유물**로 흩어져 전해지며, 인간의 기억 속에 "신화"라는 이름으로 남았습니다.

## 균열 (Rift)

시대가 흐르며 봉인이 약해질 때마다, 밤하늘의 별자리가 떨리고 **균열(Rift)** 이 열립니다. 균열은 특정 신화의 논리로 현실을 재작성합니다:

- 스틱스의 물이 강을 검게 변색
- 비프뢰스트의 무지개 다리가 성벽을 휘게 함
- 사신수가 지형을 왜곡

## 별자리의 수호자

균열이 열릴 때 **별자리의 수호자**가 깨어납니다. 이들은 신들의 연대기를 물려받은 자들로, **신화 등가 연금술(Mythic Equivalence Alchemy)** 을 통해 서로 다른 신화를 결합하는 비술을 사용합니다.

수호자의 목적:
1. 성단 제단을 수호
2. 원초의 흐름을 복원
3. 왜곡의 군대에 맞서 싸움

왜곡의 군대:
- 티아마트의 자손
- 요르문간드의 파편
- 메두사의 깨어난 석상들
- 아누비스의 방랑하는 영혼 군단
- 동방의 여우 정령들

## 승리와 패배

- **승리** — 해당 신화의 논리가 정화되어 연대기에 통합. 세계가 하나의 통합된 신화를 향해 수렴
- **패배** — 왜곡이 현실에 영구히 각인. 세계의 법칙이 변형되어 파편화 가속

## 핵심 게임 시스템

### 균열 필드 (Rift Fields)
각 전장은 하나의 신화 논리가 지배하는 "균열"이며, 고유한 맵, 웨이브 패턴, 환경 규칙을 가집니다.

### 별자리 공명 (Constellation Resonance)
타워, 영웅, 유물에 별자리 태그(사자자리, 전갈자리 등)가 부여됩니다. 같은 별자리를 맞추면 크로스 신화 시너지가 발동합니다.

### 신화 계보 시너지
같은 신화 유닛을 조합하면 교리 보너스가 적용됩니다:
- 그리스 세트 → 석화 저항 관통
- 북유럽 세트 → 냉기 슬로우 중첩
- 이집트 세트 → 처형 임계치 (아누비스의 저울)

### 유물, 의식, 각인
- **유물** — 타워에 장착하는 신화 아티팩트 (묠니르 파편, 호루스의 눈 등)
- **의식** — 맵 전체에 영향을 주는 주문 (라그나뢰크의 겨울, 불사조 의식 등)
- **각인** — 영구 지형 수정 (스틱스 절벽, 비프뢰스트 기둥 등)

### 타락 & 봉인 게이지
매 라운드마다 타락이 상승하며 엘리트 수식어가 추가되지만, 동시에 봉인 게이지도 축적됩니다. 봉인으로 보스 패턴 약화나 적 경로 변경이 가능합니다.

### 신화 연금술 (Mythic Alchemy)
서로 다른 신화의 유물을 융합하여 "등가 유물"을 생성합니다.
- 예: 헤르메스의 날개 신발 + 아누비스의 우자트 눈 = 영혼추적자의 걸음

### 균형 강제자 (흐름의 감시자)
단일 신화에 과도하게 의존하면 전역 간섭이 발동합니다. 공명 상한, 별자리 효과 반전, 시너지 일시 봉인 등으로 다양한 신화 활용을 강제합니다.
